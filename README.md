# 🛒 BlinkIT Sales Analysis Dashboard

An interactive Power BI dashboard built to analyze BlinkIT grocery sales performance using Power BI, DAX, and Power Query.

---

## 📊 Dashboard Preview

![Dashboard](BlinkIT-Sales-Analysis-Dashboard/Images/dashboard.png)

---

## 🚀 Project Overview

This dashboard provides insights into BlinkIT grocery sales by analyzing:

- 💰 Total Sales
- 📦 Number of Items Sold
- ⭐ Average Rating
- 📈 Average Sales
- 🏪 Outlet Performance
- 🥗 Fat Content Analysis
- 📍 Outlet Location Analysis
- 📦 Item Type Performance

---

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📈 KPIs

- Total Sales
- Average Sales
- Number of Items
- Average Rating

---

## 📊 Dashboard Features

- Interactive Slicers
- Dynamic KPI Cards
- Area Chart
- Donut Charts
- Bar Charts
- Matrix with Conditional Formatting
- Reset Filters Button
- Business Performance Dashboard

---

## 📌 Business Insights

- Supermarket Type1 contributes the highest overall sales.
- Fruits & Vegetables and Snack Foods are the top-performing categories.
- Tier 3 outlets generate the highest revenue.
- Medium-sized outlets contribute the largest share of sales.
- Average customer rating remains around 3.9.
